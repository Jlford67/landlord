# Landlord Web Application

A full-stack personal landlord management system to track properties, tenants, leases, and financial activity in one place. Built to replace spreadsheets and disconnected portals with a simple, reliable web app.

This project is actively developed and tailored for real-world landlord workflows.

System built with Next.js, Prisma, and PostgreSQL.
